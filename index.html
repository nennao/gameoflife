Information

This game is based on Conway's Game of Life: a living cell will "die" if it has more than 3 live neighbours, or less than 2. However, a dead cell with exactly 3 live neighbours will become live. This simple set of rules can lead to pretty complex results, and in the case of this game, pretty and complex living patterns.

The best patterns tend to be made when the initial pattern is symmetrical in some way. So go ahead and use the brushes to draw a pattern using one or more lines of symmetry as a guide. Or, choose a random pattern and click 'play' to get started!
